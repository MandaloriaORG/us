# Admin Panel

## Goal

The admin panel should allow controlling Mandaloria without touching the database manually.

Conceptual name within the product: Council.

## Panel sections

- Dashboard
- Users
- Roles and permissions
- Plazas
- Posts
- Comments
- Reports
- Moderation
- Codex Libre
- Clans/Houses
- Ranks
- Badges
- Holochat
- Reactions
- Settings
- Logs

## Dashboard

- [ ] View counts: users, posts, comments, open reports.
- [ ] View recent activity.
- [ ] View urgent reports.
- [ ] View new users.
- [ ] View recent moderation actions.

## Users

- [ ] Search users.
- [ ] Filter by status: active, suspended, banned.
- [ ] View user admin profile.
- [ ] Change display name if necessary.
- [ ] Reset avatar.
- [ ] Suspend user.
- [ ] Ban user.
- [ ] Unban user.
- [ ] View user's posts.
- [ ] View user's comments.
- [ ] View reports against the user.
- [ ] View moderation history.

## Roles and permissions

- [ ] View roles.
- [ ] Create role.
- [ ] Edit role.
- [ ] Delete role if not protected.
- [ ] Assign role to user.
- [ ] Remove role from user.
- [ ] View permissions.
- [ ] Assign permissions to role.
- [ ] Remove permissions from role.
- [ ] Configure permissions by plaza.
- [ ] Configure permissions by channel.
- [ ] Log changes in audit logs.
