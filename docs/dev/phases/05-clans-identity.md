# Phase 5 - Clans, Casas and Identity

## Objective

Give Mandaloria its own social structure.

## Clans/Casas

- [ ] List clans/casas.
- [ ] View clan/casa page.
- [ ] Create clan/casa from admin.
- [ ] Edit name, slug, description and emblem.
- [ ] Change privacy.
- [ ] Archive clan/casa.
- [ ] Request entry.
- [ ] Invite user.
- [ ] Accept request.
- [ ] Reject request.
- [ ] Leave clan/casa.
- [ ] Expel member.
- [ ] Change leader.
- [ ] Define mission.
- [ ] Assign areas of responsibility.
- [ ] Relate areas that help care for the Codex.

## Expeditions

- [ ] Create research expedition.
- [ ] Define objective, leads and participants.
- [ ] Save permitted sources.
- [ ] Statuses: proposed, active, under review, completed, cancelled and archived.
- [ ] Publish result in Plaza or propose it to the Codex.
- [ ] Protect private sources and results.

## Internal roles

- [ ] Create internal roles.
- [ ] Assign internal role.
- [ ] Remove internal role.
- [ ] Basic internal permissions.

## Ranks

- [ ] Create ranks.
- [ ] Edit ranks.
- [ ] Assign rank.
- [ ] Show rank on profile.
- [ ] Show rank next to name.

## Badges

- [ ] Create badge.
- [ ] Edit badge.
- [ ] Assign badge.
- [ ] Remove badge.
- [ ] Show badges on profile.
- [ ] Show issuer, date, reason and status.
- [ ] Link public evidence or protected reference.
- [ ] Define authorized issuers.
- [ ] Revoke without deleting internal history.

## Friends and blocks

- [ ] Send friend request.
- [ ] Accept request.
- [ ] Reject request.
- [ ] Cancel request.
- [ ] Remove friend.
- [ ] Block user.
- [ ] Unblock user.
- [ ] Respect blocks in interactions.

## Security

- [ ] Leader only manages their own clan/casa.
- [ ] Admin can intervene.
- [ ] Blocked user cannot harass with requests.
- [ ] Ranks/badges do not grant permissions without explicit rule.
- [ ] Casa responsibility does not grant ownership of the Codex.
- [ ] Private badge evidence is not exposed.

## Done when

- [ ] Users have visible identity.
- [ ] Clans/casas work.
- [ ] Friends/blocks work.
- [ ] Admin can control social structure.
